# 📝 Lista de Colores

Una aplicación web sencilla e intuitiva para gestionar y organizar colores.


## 🛠️ Tecnologías y Librerías

- **React**
- **[Bootstrap 5](https://getbootstrap.com/)**
- **[React-icons](https://react-icons.github.io/react-icons/)**

## ⚙️ Instalación y Uso

- Clona el repositorio:
   ```bash
   git clone https://github.com/nikki0011/TP-REACT-EJERCICIO-06
   ```
